# Gatotkaca Search Engine

License : MIT
<br>
Gatotkaca search engine adalah website search engine seperti pada umumnya, untuk datanya sendiri adalah hasil scraping dari goole.

</div>

## Instalation

```bash
# Clone this repo
https://github.com/bagusok/gatotkaca

# Install All Module
npm install

# Start Development
npm run dev

# Build Production
npm run build
npm start
```
